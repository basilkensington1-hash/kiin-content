## Usage Parameters
- Volume Level: 0.15
- Duck Ratio: 0.30

## File Requirements
- Format: MP3, WAV, or AAC
- Quality: 128kbps or higher
- Length: 3-10 minutes (will loop automatically)
- No vocals or prominent percussion
- Consistent volume throughout
